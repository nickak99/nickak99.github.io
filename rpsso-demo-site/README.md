# 非官方訓練示範頁

本資料夾為靜態網站示範，參考使用者提供之截圖製作舊式入口頁風格。

## 安全限制

- 本頁明確標示為「非官方示範頁」。
- 不連接任何外部登入、驗證或資料庫服務。
- 登入表單只會在瀏覽器端攔截並清空，不會送出或保存帳號密碼。
- 客服信箱、忘記密碼、諮詢電話、交流平台等連結皆為展示用途。

## 本機預覽

可在本資料夾執行：

```powershell
python -m http.server 8080
```

再開啟：

```text
http://localhost:8080/
```
